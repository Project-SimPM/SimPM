================================================================================
PROBABILITY DISTRIBUTIONS EXPLAINED
================================================================================

norm(mean, std_dev) - Normal Distribution
  Description: Bell-shaped curve, symmetric around mean
  Use: Natural variation around average (loading, travel times)
  Example: norm(7, 1.5) produces values mostly 5.5-8.5
  Parameters:
    mean: Center of distribution (most likely value)
    std_dev: Spread (smaller = concentrated, larger = spread out)

uniform(a, b) - Uniform Distribution
  Description: Flat line, all values equally likely
  Use: Bounded ranges (minimum to maximum range)
  Example: uniform(2, 4) produces any value 2-4 equally likely
  Parameters:
    a: Minimum value
    b: Maximum value

triang(a, c, b) - Triangular Distribution
  Description: Triangle shape with peak at mode
  Use: Estimate with known min, max, and most likely
  Example: triang(1, 3, 5) peaks at 3, ranges 1-5

expon(scale) - Exponential Distribution
  Description: Right-skewed, long tail
  Use: Random event times (equipment failures, arrivals)
  Example: expon(5) means ~5 average, occasional long times

Key Insight:
  Normal: For symmetric variation around an average
  Uniform: For any value in a range equally likely
  Exponential: For random event times and rare delays
