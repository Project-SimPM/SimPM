UNDERSTANDING VISUALIZATIONS (How to read and interpret)
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Time Series Graphs
  What: How a metric changes over simulation time
  Reading: X-axis = time, Y-axis = metric value
  Features:
    âœ“ Hover to see exact values
    âœ“ Range slider to zoom
    âœ“ Line graphs for continuous metrics
    âœ“ Step graphs for discrete/status metrics
  Use cases:
    â€¢ Identify when problems start
    â€¢ Track bottleneck evolution
    â€¢ Spot patterns in system behavior

Histograms
  What: Distribution/frequency of values
  Reading: X-axis = value, Y-axis = how many times that value occurred
  Shapes mean:
    â€¢ Bell curve = normal distribution (expected variability)
    â€¢ Skewed = unbalanced (many short waits, few long waits)
    â€¢ Narrow = consistent (predictable behavior)
    â€¢ Wide = variable (unpredictable behavior)
  Use cases:
    â€¢ Identify bottlenecks (cluster of long durations)
    â€¢ Check variability (spread of values)
    â€¢ Spot outliers (rare extreme values)

Box Plots
  What: Compact distribution summary using quartiles
  Reading:
    â€¢ Box = middle 50% of data
    â€¢ Line in box = median (50th percentile)
    â€¢ Whiskers = extend to min/max (or 1.5Ã— IQR)
    â€¢ Dots beyond whiskers = outliers
  Use cases:
    â€¢ Compare multiple distributions
    â€¢ Spot skewness (asymmetry)
    â€¢ Identify outliers
    â€¢ Quick visual summary

Statistics Tables
  Metrics shown:
    â€¢ Count - Number of data points
    â€¢ Mean - Average value
    â€¢ Median - Middle value (robust to outliers)
    â€¢ Min - Smallest value
    â€¢ Max - Largest value
    â€¢ Std - Standard deviation (variability measure)
  
  Time-Weighted Statistics (special for status logs):
    "Weighted by duration" means each value is weighted by how long 
    that state lasted.
    Example: If resource was busy 10 min, idle 90 min â†’ 
             Time-weighted idle = 90%
             (Not 50% like simple average would suggest)

TIPS & TROUBLESHOOTING
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Problem: No Data in a Section
  Cause: Metric not logged during simulation
  Solution: Use log=True when creating entities/resources
  Example: 
    truck = des.Entity(env, name="truck", log=True)
    loader = des.Resource(env, name="loader", log=True)

Problem: Empty Visualizations
  Cause: Selected metric has no valid numbers
  Solution: Check that data columns contain numbers (not text/missing)

Problem: Dashboard Freezes / Runs Slowly
  Cause: Very large simulations overwhelm browser
  Solutions:
    â€¢ Export data and analyze with Python/R
    â€¢ Use search/filter features
    â€¢ Run smaller scenario for prototyping

Problem: Multiple Runs - How to Compare
  Use Run Selector dropdown:
    â€¢ "All runs" = aggregate statistics across all runs
    â€¢ Specific Run ID = zoom into that run
  
  Interpretation:
    â€¢ Narrow histogram = consistent results (stable model)
    â€¢ Wide histogram = high variability (stochastic/sensitive model)

FAQ (Frequently Asked Questions)
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Q: Can I print the dashboard?
A: Yes, use browser Print. Better for PDFs: export to CSV and use 
   spreadsheet/BI tool.

Q: What if my simulation has no queues?
A: Focus on Schedule and Status Log sections. Waiting Log will be empty.

Q: How to interpret time-weighted statistics?
A: Value is weighted by duration. Resource idle 90% of time shows 90% 
   idle utilization, not 50% (which simple average would show).

Q: Can I compare two simulations side-by-side?
A: Not in dashboard directly. Export both as CSV and use Excel/Python 
   for comparison.

Q: Why so many visualizations and tabs?
A: Different questions need different views:
   â€¢ Histograms â†’ Distribution shape
   â€¢ Time series â†’ Trends and timing
   â€¢ Box plots â†’ Outliers and quartiles
   â€¢ Statistics â†’ Exact numbers
